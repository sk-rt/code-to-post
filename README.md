Code To Post - WordPress Plugin
====
## ⚠️ This plugin in development

---

- HTMLファイルから postの作成/更新
- 固定ページ・カスタムポスト対応
- ディレクトリ名を Post typeとして読み込み
- ファイル名を Post name (Slug)として読み込み
- アップデートするのは `post_content`のみ
